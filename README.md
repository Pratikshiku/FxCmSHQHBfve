### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户（员工）、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 客户信息管理
6. 设备信息管理
7. 设备维修管理
8. 设备保养管理
9. 定检信息管理
10. 配件信息管理
11. 配件采购管理


**管理员功能：**
1. 登录
2. 个人中心(修改密码、个人信息）
3. 员工管理
4. 客户信息管理
5. 设备类型管理
6. 设备型号管理
7. 设备信息管理
8. 设备维修管理
9. 设备保养管理
10. 定检信息管理
11. 配件信息管理
12. 配件采购管理
13. 合同信息管理
